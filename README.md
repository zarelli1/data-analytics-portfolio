# 🐍 Trilha Python: Algoritmos Aplicados a Mercado

### 📦 Fase 1: Fundamentos Revisados (Variáveis de Mercado)
- [ ] Tipos primitivos, listas e dicionários aplicados a dados financeiros
- [ ] Leitura e escrita de variáveis de estado (preço, volume, posição)
- [ ] Projeto: Calculadora de preço médio de compra (position averaging)
### 🗂️ Fase 2: Estruturas de Dados
- [ ] Listas, filas (queues) e pilhas (stacks)
- [ ] Complexidade de tempo: por que estrutura certa importa
- [ ] Projeto: Simulador de fila de ordens de compra/venda (order book básico)
### 🔍 Fase 3: Busca, Ordenação e Recursão
- [ ] Algoritmos clássicos de busca (linear, binária)
- [ ] Algoritmos de ordenação (bubble, merge, quick — na teoria e na prática)
- [ ] Recursão: casos base, stack de chamadas
- [ ] Projeto: Encontrar maior alta/queda em uma série histórica de preços
### ⚙️ Fase 4: Funções, Loops e Modularização
- [ ] Funções puras, parâmetros, retorno
- [ ] Loops eficientes vs loops ingênuos
- [ ] Organização de código em módulos reutilizáveis
- [ ] Projeto: Calculadora de médias móveis (SMA/EMA) do zero
### 📊 Fase 5: Manipulação de Dados (Pandas)
